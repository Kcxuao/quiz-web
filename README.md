
# Vue 在线刷题系统

一个基于 **Vue 3 + Tailwind CSS** 的在线刷题系统，支持单选题、填空题、问答题，以及题目解析、进度统计和数学公式渲染。适合作为考试练习、知识点复习或自建题库的前端模板。

---

## 功能特性

- 📄 **题目展示**
  - 支持单选题、填空题、问答题
  - 显示题目类型和难度
  - 支持 LaTeX 数学公式渲染（KaTeX）

- 📝 **答题操作**
  - 单选题点击选择答案
  - 填空题输入答案并提交
  - 自动统计正确/错误题目
  - 支持显示/隐藏题目解析及知识点标签

- 📊 **答题统计**
  - 当前正确数、错误数、正确率
  - 题目进度条展示

- 🔢 **题号导航**
  - 左侧题号栏显示所有题目
  - 不同颜色标识答题状态
  - 支持点击跳转题目
  - 可折叠/展开

- 💬 **提示信息**
  - 底部浮动提示框，自动显示操作反馈

- ⚡ **交互优化**
  - 过渡动画
  - 滚动条样式美化
  - 响应式布局

---

## 技术栈

- Vue 3（Composition API）
- Tailwind CSS
- KaTeX（数学公式渲染）
- Fetch API（加载题目 JSON）
- HTML / CSS / JavaScript

## 安装与运行

1. **克隆项目**

```bash
git clone https://github.com/Kcxuao/quiz-web.git
cd quiz-web
```

2. **安装依赖**

```bash
pnpm install
```

3. **运行项目**

```bash
pnpm dev
```

4. **访问**

> 打开浏览器访问 \`http://localhost:5173\`（Vite 默认端口）
>

---

## 使用说明

1. **选择考试与科目**  
   在顶部下拉框选择需要练习的科目，题目会自动加载。

2. **答题**  
   - 单选题：点击选项即可答题  
   - 填空题：输入答案后点击“提交答案”

3. **查看解析**  
   点击底部“显示解析”按钮查看答案解析和知识点。

4. **题号导航**  
   左侧题号栏显示每题答题状态，可点击跳转。可折叠为小按钮节省屏幕空间。

## 数据格式

题目 JSON 示例：

```json
 {
    "id": 10,
    "question": "下面哪一项不属于三三两两探讨法的步骤内容。【 】",
    "options": {
      "A": "反复思考",
      "B": "得出结果",
      "C": "交流意见",
      "D": "自由分组"
    },
    "answer": "A",
    "explanation": ""
  }
```
