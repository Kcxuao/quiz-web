# 智能答题系统

一个功能完整的在线答题平台，支持多科目题库、错题集、数据持久化等功能。适合学生复习备考和自定义试题。

## ✨ 主要功能

### 核心功能
- 📚 **多科目支持** - 支持多个考试科目和子科目的题库管理
- 📝 **多题型** - 支持单选题、填空题、问答题等多种题型
- 💾 **数据持久化** - 自动保存答题进度和状态，刷新页面后继续做题
- ❌ **错题集** - 自动收集错题，支持单独复习错题
- ⭐ **收藏功能** - 收藏重要题目，快速查看重点内容

### 学习功能
- 📊 **实时统计** - 显示正确率、错题数、正确题数
- 📈 **进度追踪** - 清晰的进度条显示做题进度
- 🎯 **难度分类** - 按难度等级（简单/中等/困难）分类题目
- 💡 **详细解析** - 每道题都有详细的解析和知识点标签
- 🔍 **智能筛选** - 支持全部题目、错题集、收藏题目快速切换

### 用户体验
- 📱 **响应式设计** - 完美适配桌面端和移动端
- 🎨 **简洁界面** - 清晰直观的UI设计，易于操作
- ⌨️ **流畅交互** - 快速的题目切换和答题反馈
- 🌐 **数学公式** - 支持使用 KaTeX 渲染数学公式

## 📋 题库数据格式

题库使用 JSON 格式存储，每个科目对应一个 JSON 文件。示例：
```json
[
  {
    "question": "什么是创新思维？",
    "type": 1,
    "options": {
      "A": "选项A",
      "B": "选项B",
      "C": "选项C",
      "D": "选项D"
    },
    "answer": "A",
    "difficulty": "easy",
    "knowledgePoints": ["创新理论", "思维方式"],
    "explanation": "创新思维是指..."
  },
  {
    "question": "填空题：________",
    "type": 2,
    "answer": "正确答案",
    "difficulty": "medium",
    "knowledgePoints": ["知识点1"],
    "explanation": "详细解析"
  }
]
```

### 字段说明
- `question` (string) - 题目内容，支持 LaTeX 数学公式（用 `$...$` 包围）
- `type` (number) - 题型：1=单选题，2=填空题/问答题
- `options` (object) - 选项（仅单选题需要），键为 A/B/C/D...，值为选项内容
- `answer` (string) - 标准答案
- `difficulty` (string) - 难度：easy/medium/hard
- `knowledgePoints` (array) - 知识点标签
- `explanation` (string) - 题目解析

## 💾 数据存储

所有数据都保存在浏览器的 `localStorage` 中：

- **答题状态**: `quiz_[考试名]_[科目名]` - 保存答题状态和进度
- **收藏数据**: `favorites_[考试名]_[科目名]` - 保存收藏的题目

数据会在以下时刻自动保存：
- 提交答案后
- 收藏/取消收藏题目后
- 答题状态改变时

## 🎮 使用指南

### 基本操作
1. **选择科目** - 在顶部选择考试和科目
2. **做题** - 点击选项选择答案或输入填空题答案
3. **查看解析** - 点击"解析"按钮查看详细解析
4. **收藏题目** - 点击右上角的 ⭐ 收藏重要题目
5. **切换题目** - 使用左侧题号栏、上下按钮或题号导航

### 筛选功能
- **全部题目** - 查看所有题目
- **❌ 错题集** - 只显示做错的题目，方便针对性复习
- **⭐ 收藏** - 只显示收藏的题目

### 移动端
- 左侧题号栏默认折叠，点击菜单按钮展开
- 展开时覆盖整个屏幕，点击遮罩或关闭按钮可隐藏
- 所有功能与桌面端保持一致

## 📊 统计指标

系统实时显示以下统计信息：
- ✅ 正确：已正确答题数
- ❌ 错误：已错误答题数
- 📊 正确率：正确题目占已答题的百分比

## 🔧 自定义配置

修改 `script setup` 中的 `exams` 和 `subSubjects` 对象可以自定义考试和科目：
```javascript
const exams = ['省考', '国考'] // 添加新的考试
const subSubjects = {
  '省考': ['创新思维', '数据结构', ...],
  '国考': ['行政能力测试', '申论', ...]
}
```

## 📱 响应式支持

- **桌面端** (≥768px) - 左侧题号栏常显示，右侧为主要答题区域
- **平板端** (768px 以下) - 题号栏可折叠，点击按钮展开为模态框
- **手机端** - 完整响应式布局，优化了移动端体验

- **数学渲染**: KaTeX
